# YosoThemeProvider

`YosoThemeProvider`는 애플리케이션에 테마를 적용하고 다크 모드를 지원하는 컴포넌트입니다.

컴포넌트를 사용하고자 하는 앱의 최상위 컴포넌트로 감싸서 사용합니다.

## Props

`theme` (선택 사항): 사용자 정의 테마를 적용하기 위한 객체입니다. `Partial<Theme>` 타입으로, 필요한 부분만 재정의할 수 있습니다.

`useSystemColorMode` (선택 사항): 시스템의 다크 모드 설정에 따라 테마를 자동으로 변경할지 여부를 설정합니다. 기본값은 false입니다.

`initialThemeMode` (선택 사항): 초기 테마 모드를 설정합니다. "light" 또는 "dark" 중 하나를 선택할 수 있으며, 기본값은 "light"입니다

## 주요 기능

- **테마 적용**: 기본 테마 또는 사용자 정의 테마를 적용할 수 있습니다.
- **다크 모드 지원**: 시스템 설정 또는 사용자 설정에 따라 다크 모드를 지원합니다.
- **테마 토글 기능**: `useTheme` 훅을 사용하여 테마 모드를 토글할 수 있습니다.

## 사용법

<br />

### 기본 사용법

yoso-ui를 사용하려면 `YosoThemeProvider`로 앱을 감싸면 됩니다.

```tsx
import { YosoThemeProvider } from "yoso-ui";

function App() {
  return (
    <YosoThemeProvider>
      <YourApp />
    </YosoThemeProvider>
  );
}

export default App;
```

### 시스템 설정에 따른 다크 모드 지원

`YosoThemeProvider`의 `useSystemColorMode` prop을 사용하여 시스템 설정에 따라 다크 모드를 지원할 수 있습니다.

```tsx
import { YosoThemeProvider } from "yoso-ui";

function App() {
  return (
    <YosoThemeProvider useSystemColorMode={true}>
      {/* 앱 내용 */}
    </YosoThemeProvider>
  );
}

export default App;
```

### 기본 테마 색상

`YosoThemeProvider`의 `initialThemeMode` prop을 사용하여 기본 테마 색상을 변경할 수 있습니다.

```tsx
import { YosoThemeProvider } from "yoso-ui";

function App() {
  return (
    <YosoThemeProvider initialThemeMode="dark">
      {/* 앱 내용 */}
    </YosoThemeProvider>
  );
}

export default App;
```

### 다크 모드 토글 기능

`useTheme` 훅을 사용하여 yoso component들의 테마 모드를 토글할 수 있습니다.

```tsx
import { useTheme } from "yoso-ui";

function YourComponent() {
  const { themeMode, toggleThemeMode } = useTheme();

  return <button onClick={toggleThemeMode}>테마 변경</button>;
}
```

### 사용자 정의 테마 적용

`YosoThemeProvider`의 `theme` prop을 사용하여 사용자 정의 테마를 적용할 수 있습니다.

```tsx
import { YosoThemeProvider, Theme } from "yoso-ui";

const customTheme: Partial<Theme> = {
  defaultColor: "#ef4567",
  // 기타 스타일 속성
};

function App() {
  return (
    <YosoThemeProvider theme={customTheme}>{/* 앱 내용 */}</YosoThemeProvider>
  );
}

export default App;
```
