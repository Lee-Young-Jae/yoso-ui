# (⚠️ 실험적) RollingNumber 컴포넌트

> ⚠️ 이 컴포넌트는 실험적인 기능으로, 사용 시 주의가 필요합니다.
>
> This component is not fully tested yet and should be used with caution.

`RollingNumber` 컴포넌트는 숫자가 부드럽게 롤링되면서 변경되는 애니메이션 효과를 제공합니다.

<br />

## 주요 기능

- **숫자 롤링 애니메이션**: 숫자가 변경될 때 부드러운 롤링 효과를 제공합니다.
- **천 단위 구분자**: `useGrouping` 옵션으로 천 단위 구분자를 표시할 수 있습니다.
- **자릿수 지정**: `digits`와 `padWithZero` 옵션으로 표시할 자릿수를 지정할 수 있습니다.
- **애니메이션 커스터마이징**: `duration`과 `timingFunction`으로 애니메이션을 조절할 수 있습니다.

<br />

## 사용 방법

```tsx
import { RollingNumber } from "yoso-ui";

const YourComponent = () => (
  <RollingNumber value={1234} duration={1} useGrouping={true} />
);
```

### 천 단위 구분자 사용

`useGrouping` 속성을 활성화하여 천 단위 구분자를 표시할 수 있습니다.

```tsx
<RollingNumber value={1234567} useGrouping={true} />
```

<Canvas of={RollingNumberStories.WithGrouping} />

### 앞자리 0으로 채우기

`digits`와 `padWithZero` 속성을 사용하여 지정된 자릿수만큼 앞자리를 0으로 채울 수 있습니다.

```tsx
<RollingNumber value={42} digits={4} padWithZero={true} />
```

<Canvas of={RollingNumberStories.PaddedNumber} />

### 동적 값 변경

상태 값과 함께 사용하여 동적으로 숫자를 변경할 수 있습니다.

```tsx
const [value, setValue] = useState(0);

<Button onClick={() => setValue(prev => prev + 599)}>
  증가 (+599)
</Button>
<RollingNumber value={value} />
```
